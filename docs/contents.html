The Parsed Corpus of Early English Correspondence 2

Letter collections

 The PCEEC2 is made up of 84 files, each containing letters from a single edited collection (or in a few cases, from two related collections). This is a subset of the collections that make up the CEEC. The collections contain the same letters as in the CEEC except for Clifford, Henslowe, Oxinden, and Royal2, which contain a subset of the original (due to copyright restrictions).
